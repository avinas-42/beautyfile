Résumé

Beauty File est un module complémentaire Google Workspace pour Google Docs. Il s’exécute dans Google Apps Script lorsque vous utilisez l’add-on dans l’interface Docs. Il n’existe pas de serveur « Beauty File » distinct qui collecterait vos documents : les traitements liés à l’add-on s’effectuent dans l’infrastructure Google lorsque vous utilisez le produit.

Données traitées

  • Le contenu du document Google Docs ouvert lorsque vous utilisez l’add-on, dans la mesure nécessaire pour appliquer la mise en forme que vous demandez dans le panneau.
  • Les préférences stockées par le script pour votre compte (par exemple les dernières options sélectionnées), via les mécanismes Google Apps Script (propriétés utilisateur).

Finalités

Appliquer les options de mise en forme (portée des lettres, police, taille, gras, couleur) que vous déclenchez dans l’add-on sur le document ouvert.

Conservation

Beauty File ne maintient pas une base de données dédiée hors Google pour le contenu de vos documents. La conservation du document et des révisions relève de votre compte Google et des conditions applicables à Google Workspace / Google Docs.

Partage avec des tiers

L’add-on ne transfère pas vos documents vers un backend Beauty File. Si vous ajoutez ultérieurement des services tiers (analytique, API externes, etc.), cette page sera mise à jour en conséquence.
